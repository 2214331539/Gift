# 05. 数据接口设计

## 1. API 约定

基础路径：`/api/v1`

请求格式：`application/json`

认证方式：

- Web 使用 HttpOnly Cookie 保存会话。
- 管理端接口必须校验角色。
- 图片上传使用后端签发的对象存储上传凭证。

统一响应：

```json
{
  "data": {},
  "meta": {},
  "error": null
}
```

统一错误：

```json
{
  "data": null,
  "meta": {},
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "请求参数不合法",
    "details": []
  }
}
```

分页参数：

| 参数 | 说明 |
| --- | --- |
| `page` | 页码，从 1 开始 |
| `pageSize` | 每页数量，默认 20，最大 50 |
| `cursor` | 游标分页场景使用 |

## 2. 鉴权接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `POST` | `/auth/register` | 邮箱注册 |
| `POST` | `/auth/login` | 登录 |
| `POST` | `/auth/logout` | 退出 |
| `GET` | `/auth/me` | 当前登录用户 |

登录请求：

```json
{
  "email": "user@example.com",
  "password": "password"
}
```

## 3. 礼物故事接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/stories` | 礼物列表、搜索、筛选 |
| `GET` | `/stories/:id` | 礼物详情 |
| `POST` | `/stories` | 创建礼物故事 |
| `PATCH` | `/stories/:id` | 更新草稿或被驳回内容 |
| `DELETE` | `/stories/:id` | 作者删除或隐藏 |
| `POST` | `/stories/:id/submit` | 提交审核 |
| `POST` | `/stories/:id/view` | 记录浏览 |

列表查询参数：

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `q` | string | 自然语言或关键词 |
| `recipientType` | string | 送给谁 |
| `occasion` | string | 场景 |
| `budgetRange` | string | 预算 |
| `giftType` | string | 礼物类型 |
| `emotion` | string | 情绪 |
| `interest` | string | 兴趣 |
| `sort` | string | 排序方式 |

创建礼物故事：

```json
{
  "title": "给考研结束的她做了一个重启生活礼盒",
  "recipientType": "girlfriend",
  "relationshipStage": "1-3_years",
  "occasion": "exam_finished",
  "budgetRange": "100_300",
  "preparationTime": "3_7_days",
  "giftTypes": ["diy", "comfort"],
  "description": "礼盒里有手写卡、茶包和周末出游计划。",
  "reason": "她考研结束后很空，我想让她重新期待生活。",
  "reaction": "她拆到最后一张卡片时哭了。",
  "tips": "内容比盒子本身更重要。",
  "notSuitableFor": "不喜欢仪式感的人可能无感。",
  "tagIds": ["tag_1", "tag_2"],
  "imageIds": ["img_1"],
  "coverImageId": "img_1",
  "isAnonymous": true,
  "privacy": {
    "hideExactPrice": true,
    "hideRelationshipDuration": false
  }
}
```

## 4. 图片接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `POST` | `/media/upload-token` | 获取上传凭证 |
| `POST` | `/media/complete` | 上传完成回调 |
| `PATCH` | `/media/:id` | 更新 alt、裁剪信息 |
| `DELETE` | `/media/:id` | 删除未使用图片 |

上传完成请求：

```json
{
  "objectKey": "stories/2026/05/sample-gift.jpg",
  "fileName": "gift.jpg",
  "mimeType": "image/jpeg",
  "size": 1200344,
  "width": 1200,
  "height": 900
}
```

## 11. 错误码

| code | 说明 |
| --- | --- |
| `UNAUTHORIZED` | 未登录 |
| `FORBIDDEN` | 无权限 |
| `VALIDATION_ERROR` | 参数校验失败 |
| `NOT_FOUND` | 资源不存在 |
| `CONFLICT` | 重复操作 |
| `RATE_LIMITED` | 请求过快 |
| `CONTENT_UNDER_REVIEW` | 内容审核中 |
| `CONTENT_REJECTED` | 内容被驳回 |
| `UPLOAD_FAILED` | 文件上传失败 |
| `INTERNAL_ERROR` | 服务器错误 |
